# 📊 Customer Retention & Churn Analysis

## 🔍 Project Overview

This project analyzes customer churn behavior in a telecom dataset to identify patterns, retention drivers, and actionable insights to reduce customer loss.

---

## 🎯 Objectives

* Understand why customers churn
* Identify high-risk customer segments
* Analyze customer lifetime patterns
* Provide business recommendations to improve retention

---

## 🛠️ Tools Used

* Power BI
* Excel

---

## 📊 Dashboard Features

* KPI Metrics (Total Customers, Churn Rate)
* Churn Distribution Analysis
* Contract Type vs Churn
* Tenure Group Analysis
* Payment Method Insights
* Interactive Filters (Slicers)

---

## 🔍 Key Insights

* Month-to-month contract customers show the highest churn
* Most churn occurs in the early stages (0–1 year)
* Electronic check users have higher churn rates

---

## 💡 Recommendations

* Offer incentives for long-term contracts
* Improve onboarding experience for new customers
* Target high-risk customers with personalized offers
* Encourage automatic payment methods

This project demonstrates how data-driven insights can help businesses reduce churn and improve customer retention.
